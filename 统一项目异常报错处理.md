# 统一项目异常报错处理

## Core Features

- 创建自定义业务异常类 BusinessException

- 实现全局异常过滤器 HttpExceptionFilter

- 使用 error-codes.ts 集中管理所有业务错误

- 重构项目中不规范的异常抛出逻辑

- 补充产品相关错误码

## Design

通过创建 BusinessException 自定义异常类和 HttpExceptionFilter 全局过滤器，实现了统一的异常处理机制。所有业务异常都通过 ERROR_CODES 进行管理，确保错误信息的一致性和可维护性。
